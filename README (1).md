# Projet de Programmation Avancée

## Cadre du projet
Ce projet a été réalisé dans le cadre du cours de Programmation Avancée, visant à mettre en pratique les concepts avancés de programmation à travers le développement d'une application web d'analyse économique. L'objectif était de concevoir et implémenter une solution logicielle robuste en appliquant les meilleures pratiques de développement pour analyser les indicateurs économiques de l'Algérie.

## Module concerné
- Programmation Avancée
- Spécialité : BigData Analytics
- Semestre : Semestre 1
- Année universitaire : 2025/2026

## Concepts clés utilisés

### 1. Traitement des données
   - Recherche de sources de données fiables
   - Collecte et agrégation de données
   - Transformation et conversion de formats vers JSON
   - Stockage structuré dans bases de données SQL

### 2. Visualisation des données
   - Extraction ciblée via requêtes HTTP
   - Représentation graphique avec Chart.js
   - Création des charts interactifs

### 3. Architecture logicielle
   - Développement backend avec Flask (Python)
   - Conception d'API RESTful
   - Intégration de bibliothèques spécialisées (JSON, MySQL Connector)

### 4. Interface utilisateur
   - Développement frontend avec HTML/CSS/JavaScript
   - Intégration de bibliothèques de visualisation (Chart.js)
   - Développement d'interfaces responsives

### 5. Intégration système
   - Communication client-serveur via API
   - Échange de données asynchrone (fetch API)
   - Format d'échange standardisé (JSON)

## Questions d'analyse posées

### 1. Quelles sont les performances économiques de l'Algérie sur les 20 dernières années ?
  
### 2. Quelle est l'évolution du PIB en Algérie ?
  
### 3. Quelle est l'évolution de l'inflation et son impact sur le pouvoir d'achat ?
  
### 4. Quelle est l'évolution du prix du pétrole en Algérie ?

### 5. Existe-t-il une corrélation entre pétrole, inflation et PIB ?

## Ce que nous avons appris à travers ce projet

### Compétences techniques développées

### 1. Maîtrise des technologies web complètes
   - Intégration réussie d'une pile technologique complète (Frontend, Backend, Base de données)
   - Expérience pratique avec Flask pour créer des API RESTful robustes
   - Utilisation avancée de JavaScript modern avec fetch API

### 2. Traitement et transformation des données
   - Conversion et normalisation des données vers JSON
   - Conception de schémas de base de données optimisés pour les données JSON
   - Implémentation de données de la collecte à la visualisation

### 3. Visualisation de données interactive
   - Maîtrise de Chart.js pour créer des visualisations dynamiques
   - Adaptation des représentations graphiques aux différents types de données
   - Création d'interfaces utilisateur intuitives pour l'exploration des données

### 4. Recherche et évaluation des sources de données
   - Identification de sources de données fiables et bénéfiques pour obtenir des insights pertinents
   - Recherche de données stratégiques ayant une réelle valeur ajoutée métier
   - Collecte de ces données via des APIs ou par téléchargement direct

### 5. Compétences transversales acquises
   - Résolution de problèmes complexes de communication client-serveur
   - Débogage efficace d'applications web complètes
   - Optimisation des performances des requêtes et du rendu
   - Adaptabilité face aux changements de spécifications et aux données réelles

### 6. Analyse économique concrète
   - Compréhension des indicateurs macroéconomiques : PIB, inflation, pouvoir d'achat
   - Application des concepts statistiques : Moyenne, variance, corrélation, normalisation des données
   - Interprétation des données temporelles : Identification de tendances, Identification des crises

### 7. Contextualisation des résultats
   - Analyse de l'économie algérienne : Compréhension de sa structure rentière et dépendance aux hydrocarbures
   - Évaluation de la volatilité économique : Mesure de l'impact des chocs pétroliers (2014, 2020) sur la croissance
   - Corrélation des variables : Démonstration des liens entre prix du pétrole, PIB et inflation

### 8. Visualisation adaptée au domaine
   - Sélection de graphiques appropriés : Courbes chronologiques pour les tendances, diagrammes en barres pour les comparaisons, nuages de points pour les corrélations
   - Représentation sectorielle : Utilisation de diagrammes circulaires pour la composition du PIB par secteur
   - Analyse comparative : Visualisation normalisée pour comparer des séries avec différentes échelles

### 9. Méthodologie d'analyse structurée
   - Formulation de questions pertinentes : Définition d'objectifs d'analyse clairs et mesurables
   - Application de méthodes statistiques : Utilisation de calculs appropriés pour répondre à chaque question
   - Interprétation critique : Évaluation des résultats dans leur contexte économique 

### 10. Synthèse et communication
   - Rédaction de rapports techniques : Structuration d'analyses complexes en sections cohérentes
   - Présentation des résultats : Articulation claire des conclusions

### 11. Compréhension approfondie du cas algérien
   - Dépendance aux hydrocarbures : Confirmation de la forte corrélation entre prix du pétrole et performance économique
   - Volatilité structurelle : Identification de la sensibilité aux chocs externes
   - Défis de diversification : Analyse des obstacles à la transformation structurelle de l'économie
